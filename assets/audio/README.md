# Audio assets

## `backing-track.mp3`

User-supplied “Borrowed Memories” exploration score by ava_r_zhou. The game initializes it on the title page as a continuous looping Web Audio source. Its dedicated music bus remains steady while the stronger effects bus keeps inspection, restoration, timeline, contradiction, phase, transition, and ending cues clear above it. If audible autoplay is blocked by the browser, playback resumes with the first permitted user gesture.
